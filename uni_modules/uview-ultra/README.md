全面兼容uni-app/uni-app-x/nvue/鸿蒙的uni-app生态框架，丰富的组件和便捷的工具以及兼容的API会让您轻松升级，事半功倍。

## 使用方法
配置easycom规则后，自动按需引入，无需`import`组件，直接引用即可。

```html
<template>
	<up-button text="按钮"></up-button>
</template>
```
